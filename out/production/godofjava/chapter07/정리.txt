1. 배열을 선언할 때 어떤 키워드를 타입 앞이나 뒤에 사용해야 하나요? []
2. 배열의 첫번째 위치는 0인가요? 1인가요? 0부터 시작.
3. 배열을 선언할 때 boolean 배열의 크기만 지정했다면 boolean 배열의 [0] 위치에 있는 값은 무엇인가요? primitive type의 경우 배열의 크기만 지정하면 기본값으로 초기화된다. 따라서 boolean배열의 [0]위치에 있는 값은 false가 된다.
4. ArrayIndexOutOfBoundException 이라는 것은 언제 발생하나요? Array의 index를 벗어난 범위의 인덱스를 참조하거나, 값을 할당하려고 할때 발생한다. runtime exception으로 컴파일을 되지만 해당 부분 실행시 에러가 발생한다.
5. 중괄호를 이용하여 배열을 초기화할때 중괄호 끝에 반드시 어떤 것을 입력해주어야하나요? 배열의 선언과 동시에 초기화할때 {}를 통해 직접 값을 대입할 수 있다. 이때 사용되는 중괄호는 블럭을 의미하는 중괄호와 다른 의미이므로 세미콜론으로 한 줄의 끝임을 알려주어야한다.
6. 2차원 배열을 정의할 때에는 대괄호를 몇 개 지정해야하나요? [][] 2개
7. 배열을 쉽게 처리해주는 for문의 문법은 어떻게 되나요? 배열의 인덱스 없이 배열의 각 원소만 참조하면 될때 사용할 수 있으며 for(원소타입 임시변수명: 순회할 배열 변수)로 사용한다
8. 자바 프로그램에 데이터를 전달해 주려면 클래스 이름 뒤에 어떻게 구분하여 나열하면 되나요? 공백으로 띄어서 나열하면 main 함수에 String 배열로 전달된다.
9. 자바 프로그램이 시작할때 전달 받는 내용은 어떤 타입의 배열인가요? String 타입 배열.
