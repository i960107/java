1.순서와 상관없는 여러 데이터를 하나의 객체에 저장할때 사용하는 Collection의 하위 인터페이스는? Set
2.HashSet 클래스는 생성자를 통하여 저장 가능한 데이터의 초기 크기를 지정할 수 있나요? yes HashSet(int capacity)
3.HashSet 클래스에서 객체에 데이터를 추가하는 메소드는? add()
4.HashSet 클래스에서 객체에 어떤 데이터가 존재하는지 확인하는 메소드는? contains(E e)
5.HashSet 클래스의 객체에 어떤 데이터를 삭제하는 메소드는? remove(E e)
6.Queue는 FIFO를 처리하기 위한 인터페이스입니다. FIFO는 무슨 단어의 약자인가요? First in First Out
7.Deque는 무슨 단어의 약어이며, 용도는 무엇인가요? Double ended Queue. Queue이면서 Stack. 데이터의 앞이나 뒤에서 삭제하는 것이 똑같이 O(1).
8.LinkedList 클래스의 특징을 이야기 해 보세요. 데이터에 순서가 있다. List와 Queue인터페이스를 상속받는다. 데이터의 추가와 삭제가 빠르다. 검색이 느리다.
